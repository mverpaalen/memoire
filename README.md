# Base de connaissances — Mémoire

Dossier de fiches Markdown (compatible Obsidian) construit à partir de l'export Notion "L’ENSCI - Les Ateliers".
Ouvre ce dossier comme un vault dans Obsidian (ou tout éditeur supportant les liens [[wikilinks]]) pour naviguer d'une fiche à l'autre.

## Structure

- **plan** (25 fiches) — le squelette du mémoire, section par section. Voir [[plan/00 - Vue d'ensemble du plan|Vue d'ensemble du plan]] pour la version linéaire avec tous les liens.
- **expériences et cas** (27 fiches) — les cas concrets (missions, projets, cours) mobilisés comme matière empirique.
- **références et concepts** (20 fiches) — lectures, citations, cadres théoriques.
- **idées, thèses et arguments** (2 fiches) — les hypothèses et thèses en cours de maturation.
- **annexes** — fichiers joints (PDF, etc.) référencés par les fiches.

## Comment les fiches sont reliées

Chaque fiche affiche en tête ses relations vers les autres catégories (par ex. une fiche de Plan liste ses Expériences et Références ; une Expérience liste le Plan et les Idées qu'elle nourrit). Cliquer sur un `[[lien]]` ouvre directement la fiche correspondante — exactement la logique des bases liées dans le Notion d'origine, portée dans des fichiers texte.

Pour écrire le mémoire : pars de la Vue d'ensemble du plan, ouvre chaque section, et remonte vers les Expériences / Références / Idées qui lui sont rattachées pour rédiger le contenu.
